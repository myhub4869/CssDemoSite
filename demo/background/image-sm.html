ボックスサイズ＝画像サイズ

コード内容

HTML側
<div class="box-a"></div>
CSS側
.box-a  {
	width: 250px;
	height: 200px;
	backgound-image: url("demo-image.png");
}

表示結果

ボックスサイズの横幅だけ画像サイズの2倍
